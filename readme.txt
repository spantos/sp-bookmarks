Installation Instructions
1.	Copy this repo into one of your apps directory.
2.	Create a MySQL database.
3.	In the file config.php enter the appropriate information for the database.
a.	MySQL database username
b.	MySQL database password
c.	MySQL hostname

Configuration instruction
Certain settings can be made by changing the parameter in config.php file.
In classes SendMail and SmtpEmail  enter appropriate value for SMTP host.
